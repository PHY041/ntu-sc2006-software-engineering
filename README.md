# SC2006 Software Engineering (NTU Y2S2)

Coursework archive from Nanyang Technological University (NTU), Singapore.

## What's in here

This repository contains lecture references, assignment code, project work, and personal notes accumulated while taking this module. It's published as a study reference for current and future NTU juniors.

## Important disclaimers

**Lecture materials & assignment briefs**: All slide decks, problem sheets, sample answers, and reference materials in this repository are the property of NTU and the respective course instructors. They are shared here for educational reference only. If you are an instructor or rights holder and wish for any material to be removed, please open an issue and it will be taken down promptly.

**Personal identifiers**: Filenames and text content have been redacted of student matriculation numbers and names. Some PDF / DOCX files may still retain author metadata in their internal headers. If you spot a leak, please open an issue.

**Code**: Student-authored code is released under MIT.

**Big files**: Datasets, model checkpoints (.h5 / .pt), and any single file >50MB have been excluded — see download links / instructions inside the relevant subfolder where applicable, or recreate them by re-running the project setup.

## Using this responsibly

If you're a junior taking this module:

- Read, learn, adapt patterns
- **Do not copy submissions verbatim** — NTU's plagiarism detection is robust, and you'll learn nothing
- Open an issue if something is unclear
